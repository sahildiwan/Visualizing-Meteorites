Visualizing Meteorites

The comprehensive data set to be used is from The Meteoritical Society and contains information on all of the known meteorite landings. An interactive map for data visualizations, along with a sortable bar chart will be used to shows all known meteorite landings with recorded locations.

Proposal

I will be creating two visualizations using the data set I have found. The first will be a world map view which will show all known meteorite landings, with markers scaled by meteorite mass. The second will be a sortable bar chart displaying the number of meteorites that have fallen per half-century or so.

Large problems to tackle:

  • Convert data set to CSV or JSON
  • Figure out how to visualize data set in interactive map view
  • Figure out how to visualize in interactive sortable bar chart
