Proposal

I have created two visualizations using the described data set found. The first is an interactive world map view which shows all known meteorite landings, with markers scaled by meteorite mass. The second is a sortable bar chart displaying the number of meteorites that have been found per decade.

Large problems tackled:

  • Converted data set to JSON, correctly
  • Figured out how to visualize data set in interactive map view
  • Figured out how to visualize data set in sortable bar chart

    The Data

    The Meteoritical Society records all known meteorites in its Meteoritical Bulletin and publishes Meteoritics and Planetary Science, a leading journal of planetary science.

    The comprehensive data set used is from The Meteoritical Society and contains information on all of the known meteorite landings around the world. The Fusion Table was collected by Javier de la Torre and provided in an XLS file that consisted of 34,513 meteorites with the following fields:

  • Name
  • NameType
  • RecClass
  • Mass
  • Year
  • ID
  • RecLat
  • RecLong

    The Technologies

    Three major web technologies/frameworks were used to complete this project.

  • D3.js is a JavaScript library for manipulating documents based on data. D3 visualizes data using HTML, SVG (Scalable Vector Graphics), and CSS. D3â€™s emphasis on web standards allows for full capabilities of modern browsers, combining powerful visualization components and a data-driven approach to DOM manipulation. Created and maintained by some smart people in the Stanford Visualization Group.
  • DataMaps is a framework used to create interactive maps for data visualizations. DataMaps is intended to provide data visualizations based on geographical data. It is SVG-based, can scale to any screen size, and includes everything inside of a single script file. It heavily relies on the D3.js library. Created and maintained by Mark DiMarco.
  • The site itself was created by using Bootstrap. Bootstrap is a sleek, intuitive, and powerful mobile first front-end framework for faster and easier web development. Created and maintained by some smart people at Twitter.
